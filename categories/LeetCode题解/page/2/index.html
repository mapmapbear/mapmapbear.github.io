1.题目解释

给你一个大小为 m x n 的矩阵 mat ，请以对角线遍历的顺序，用一个数组返回这个矩阵中的所有元素。

示例 1：

示例 2：

提示：

2.思路解释

对角线的个数为 n + m - 1条(n为行数, m为列数)

由于是从0开始计数,所以最后一条对角线为n+m-2,

当对角线序号为单数的时候,是从上往下遍历,反之从下往上遍历

对角线序号为 n + m - 1条,可是行数只有n行,所以在从上往下遍历的时候的,行号的上区间为min(i, n -1),行号的下区间为max(0, 1-m+1)

1 - m + 1的推导过程:

假设现在是在最后一条对角线所表示的元素,此时i的取值为n + m - 2, 再看现在行号的取值为n - 1

行号的推导公式为: j = n - 1 - (n + m - 2 - i); 将i = n + m - 2代入是 j = n -1

公式化简为:j = i - m + 1

正对角线的性质:同一对角线上的坐标之和相等

所以直接将[j, i - j]弹入答案数组

从下往上遍历的时候直接将刚才的过程反过来就可以了

LeetCode 1991.找到数组的中间位置

1.题目解释

给你一个下标从 0 开始的整数数组 nums ，请你找到 最左边 的中间位置 middleIndex （也就是所有可能中间位置下标最小的一个）。

中间位置 middleIndex 是满足 nums[0] + nums[1] + … + nums[middleIndex-1] == nums[middleIndex+1] + nums[middleIndex+2] + … + nums[nums.length-1] 的数组下标。

如果 middleIndex == 0 ，左边部分的和定义为 0 。类似的，如果 middleIndex == nums.length - 1 ，右边部分的和定义为 0 。

请你返回满足上述条件 最左边 的 middleIndex ，如果不存在这样的中间位置，请你返回 -1 。

示例 1：

输入：nums = [2,3,-1,8,4]
输出：3
解释：
下标 3 之前的数字和为：2 + 3 + -1 = 4
下标 3 之后的数字和为：4 = 4
示例 2：

输入：nums = [1,-1,4]
输出：2
解释：
下标 2 之前的数字和为：1 + -1 = 0
下标 2 之后的数字和为：0
示例 3：

输入：nums = [2,5]
输出：-1
解释：
不存在符合要求的 middleIndex 。
示例 4：

输入：nums = [1]
输出：0
解释：
下标 0 之前的数字和为：0
下标 0 之后的数字和为：0

2.思路解释

求出所有的总和total,然后在idx这个位置分为前后两个部分,前缀和(lSum),后缀和(total - lSum - nums[idx])

前缀和和后缀和相等的时候,返回此时的idx的下标

LeetCode 240.搜索二维矩阵II

1.题目解释

编写一个高效的算法来搜索 m x n 矩阵 matrix 中的一个目标值 target 。该矩阵具有以下特性：

每行的元素从左到右升序排列。
每列的元素从上到下升序排列。

示例 1：

输入：matrix = [[1,4,7,11,15],[2,5,8,12,19],[3,6,9,16,22],[10,13,14,17,24],[18,21,23,26,30]], target = 5
输出：true
示例 2：

输入：matrix = [[1,4,7,11,15],[2,5,8,12,19],[3,6,9,16,22],[10,13,14,17,24],[18,21,23,26,30]], target = 20
输出：false

2.思路解释

从右上角开始遍历,因为整个矩阵的局部单调性的特性,如果当前的右上角的位置等于target的话直接返回,如果大于表示当前的列中所有的值都大于target,只能存在于前面列,如果当前值小于target,表示当前行最大值都小于target只能在下面行中才存在匹配项

3.代码题解

时间复杂度:O(m + n) 从右上角找到左下角

空间复杂度:O(1)

LeetCode 520.检测大写字母

我们定义，在以下情况时，单词的大写用法是正确的：

全部字母都是大写，比如 “USA” 。
单词中所有字母都不是大写，比如 “leetcode” 。
如果单词不只含有一个字母，只有首字母大写， 比如 “Google” 。
给你一个字符串 word 。如果大写用法正确，返回 true ；否则，返回 false 。

示例 1：

输入：word = “USA”
输出：true
示例 2：

输入：word = “FlaG”
输出：false

2.思路解释

满足单词大写拼写标准的条件

  • 全是小写,没有大写
  • 第一个是大写,其他全是小写
  • 全是大写

小写单词的字母数是一个很关键的条件: size(全是小写), 0(全是大写), size-1(可能第一个是大写?)

不满足上述条件的单词都不符合条件返回false,满足的返回true
