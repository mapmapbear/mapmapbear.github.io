给你两个字符串 s 和 goal ，只要我们可以通过交换 s 中的两个字母得到与 goal 相等的结果，就返回 true ；否则返回 false 。

交换字母的定义是：取两个下标 i 和 j （下标从 0 开始）且满足 i != j ，接着交换 s[i] 和 s[j] 处的字符。

LeetCode 344.反转字符串

1.题目解释

编写一个函数，其作用是将输入的字符串反转过来。输入字符串以字符数组 s 的形式给出。

不要给另外的数组分配额外的空间，你必须原地修改输入数组、使用 O(1) 的额外空间解决这一问题。

示例 1：

输入：s = [“h”,”e”,”l”,”l”,”o”]
输出：[“o”,”l”,”l”,”e”,”h”]
示例 2：

输入：s = [“H”,”a”,”n”,”n”,”a”,”h”]
输出：[“h”,”a”,”n”,”n”,”a”,”H”]

提示：

1 <= s.length <= 105
s[i] 都是 ASCII 码表中的可打印字符

2.思路解释

方法1: 直接用STL中的reverse算法

方法2: 双指针运算,头指针指向第一个字符,尾指针指向最后一个字符,每次指针++的时候交换指针所指的值

2.思路解释

分两种情况: 奇数的回文串 偶数的回文串

枚举原字符串的每个字符, 分别将两个指针l, r指向回文串的两端

将满足条件的字串返回弹入res
